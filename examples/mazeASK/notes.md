# mazeASK Notes

`mazeASK` is the motivating ASK project for this repo's first structure.

In this mapping:

- the target repo keeps its own repo-local execution rules in `AGENTS.md`
- the control-surface artifact sits outside the target repo and governs the workflow around it
- startup prompts attach ChatGPT and Codex to the target repo
- this repo publishes reusable assets distilled from that pattern
